Responsive Landing Page

A modern, fully responsive landing page built using HTML, CSS, and JavaScript. It delivers a smooth user experience with a clean UI, interactive navigation, and well-structured sections.

Overview

This project demonstrates front-end development skills by creating a multi-section landing page with a dynamic navigation bar, engaging hero section, and organized content layout.
 
 Features
 
Responsive design for all screen sizes (mobile, tablet, desktop)
Fixed navigation bar with scroll-based styling
Smooth scrolling navigation
Active section highlighting
Interactive hamburger menu for mobile devices
Modern UI with gradients, animations, and clean layout
Structured sections: Home, About, Projects, Contact

Tech Stack

HTML5 – Structure and layout
CSS3 – Styling, Flexbox, Grid, animations
JavaScript (ES6) – Interactivity and DOM manipulation
 
Functionality

Navbar Scroll Effect:-
Changes appearance when the user scrolls down
Smooth Navigation:-
Scrolls smoothly to selected sections
Active Link Highlighting:-
Highlights the current section in the navbar
Mobile Menu Toggle:-
Hamburger menu for smaller screens

Getting Started

1. Clone or download the repository
2. Ensure all files are in the same directory
3. Open index.html in any modern web browser

Learning Outcomes

Building responsive layouts using CSS
Implementing interactive UI with JavaScript
Structuring a complete front-end project
Enhancing user experience with smooth navigation
